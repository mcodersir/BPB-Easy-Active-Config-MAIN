# شروع سریع نسخه ۲

۱. فایل ZIP را Extract کن.
۲. روی `run_windows.bat` یا `run_mac_linux.sh` بزن.
۳. اگر Worker نداری، API Token و Account ID را وارد کن و Deploy را بزن.
۴. لینک Subscription را در مرحله ۲ بگذار.
۵. حالت Nova Easy را دست نزن و فقط Start را بزن.
۶. خروجی اصلی را از این فایل بردار:

```text
output/nova_best_config_only.txt
```

اگر جواب نگرفت، فایل زیر را import کن و چند گزینه بعدی را تست کن:

```text
output/nova_working_configs.txt
```
