راهنمای دقیق Cloudflare

  1. روی «ثبت‌نام Cloudflare» بالا بزن و وارد حساب شو.
  2. به API Tokens برو:
  3. Create Token را بزن.
  4. Permission: Account → Workers Scripts → Edit
  5. توکن را اینجا Paste کن و «تست توکن» را بزن.
  6. اگر Account ID خودکار نیامد، از Workers & Pages → Account details کپی کن.
